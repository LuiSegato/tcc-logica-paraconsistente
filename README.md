# Priorização de Projetos com Lógica Paraconsistente

Trabalho de Conclusão de Curso apresentado à Universidade Virtual do Estado de São Paulo (UNIVESP).

## Objetivo

Desenvolver uma ferramenta de apoio à decisão para priorização de projetos utilizando lógica paraconsistente.

## Tecnologias

- Python
- Pandas
- NumPy
- Streamlit
- Matplotlib

## Funcionalidades

- Importação de planilhas Excel
- Cálculo de μ e λ
- Grau de Certeza (G)
- Grau de Contradição (C)
- Classificação dos projetos
- Dashboard interativo
- Comparação com Média Simples

## Execução

Instale as dependências:

```bash
pip install -r requirements.txt
```

Execute:

```bash
streamlit run app.py
```
